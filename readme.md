# Fluvius - open the stream

A site that shows all you relevent streams. First idea: rss streams.

# development

* source set_fluvius_envs.sh
* cd src/fluvius/ && go get
* go install fluvius
* bin/fluvius
* http://localhost:8080/

# test
* cd src/fluvius
* go test
